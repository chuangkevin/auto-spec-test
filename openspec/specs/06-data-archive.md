# 功能模組：資料歸檔（規格書庫）

## 規格書列表頁
篩選條件：產品（下拉選單）

列表欄位：
| 欄位 | 說明 |
|------|------|
| 規格書檔案 | 顯示檔名，點擊可下載原始檔案 |
| 測試腳本 | 點擊可下載 .md |
| 所屬專案 | 點擊可跳轉至專案頁面 |
| 產品名稱 | 歸屬的產品 |
| 上傳人員 | 上傳者名稱 |
| 更新日期 | 最後更新時間 |

## 卡片操作按鈕
- **更新**：跳轉至該專案的測試腳本 Tab，進行重新上傳與產出流程
- **刪除**：二次確認後刪除規格書及對應腳本

## 對應關係
- 一份規格書對應一份測試腳本（一對一）
- 刪除規格書時同步刪除對應腳本（需確認提示）
